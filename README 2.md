> **Хакатон:** Tech Vision 2026 · направление **Social & Human Capital** · Назарбаев Университет
> **Команда:** Ақнұр (код) + девочки (презентация/исследование)
> **Дедлайн онлайн-этапа:** 21 июля 2026, 23:59 · **Финал:** 27 июля 2026
> **Написано:** 19 июля 2026 · **Окно до онлайн-дедлайна: ~2.5 дня**

Это проект команды Ақнұр на республиканский молодёжный хакатон. Кейс №10 — «Гражданская
грамотность» (Civic Rights & Literacy): официальные документы, законы и госуслуги написаны
языком для юриста, а не для 18-летнего, который впервые устраивается на работу, снимает
квартиру или открывает ИП.

**Правило хакатона №1 (строгое):** один пользователь — одна острая боль. Жюри проверяет
**глубину** одной находки, а не широту. Поэтому мы НЕ делаем «энциклопедию прав». Мы делаем
один инструмент для одного человека с одной болью.

## Что мы делаем (рекомендация — см. VISION.md)

**Пользователь:** молодой человек 16–23 лет, устраивается на **первую работу** (летняя
подработка, первый оффер).
**Боль:** он не понимает, что законно, а что нет, в первом трудовом договоре — и либо молча
терпит (работа без договора, неоплаченные переработки, задержка зарплаты, «испытательный
срок» без границ, увольнение одним днём), либо узнаёт о своих правах слишком поздно.

**Инструмент (MVP):** веб-приложение, где человек своими словами описывает свою ситуацию на
русском/казахском, а ИИ-ассистент:
1. понимает ситуацию;
2. находит **конкретные статьи Трудового кодекса РК** в нашей базе знаний;
3. объясняет права **простым языком** (это и есть боль — язык);
4. даёт **конкретные следующие шаги** («что сказать работодателю», «куда обратиться»);
5. **ссылается на точную статью** — чтобы это был не «додуманный ChatGPT», а обоснованный ответ.

Полная логика и почему это сильно по критериям жюри — в [BUILD-PLAN.md](BUILD-PLAN.md).

## Стек (кратко — детали в BUILD-PLAN)

- **Frontend + Backend:** Next.js (React) — один репозиторий, деплой в один клик на Vercel
- **ИИ:** Claude API (Anthropic) — `claude-sonnet-5` для ответа, `claude-haiku-4-5` для дешёвого шага
- **База знаний:** `knowledge-base.json` — ~20 статей ТК РК, простым языком, с ссылками на adilet.zan.kz
- **Хостинг/демо:** Vercel (даёт живую https-ссылку — это и есть «demo MVP» для сдачи)

## Как запустить локально (заполнить, когда будет код)

```bash
# 1. Установить зависимости
npm install

# 2. Создать .env.local и вписать ключ Anthropic (НЕ коммитить!)
echo "ANTHROPIC_API_KEY=sk-ant-..." > .env.local

# 3. Запустить дев-сервер
npm run dev
# → открыть http://localhost:3000
```
